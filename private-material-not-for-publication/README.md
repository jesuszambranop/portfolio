# Material privado / no publicable

Usa esta carpeta solo como recordatorio local. No subas aquí piezas sensibles si el repositorio será público.

No publicar:
- Campañas de confrontación política o ataque.
- Casos con nombres de terceros sin autorización.
- Métricas internas de clientes.
- Capturas de crisis, acusaciones o conflictos.
- Contenido que pueda interpretarse como difamatorio o agresivo.

Si necesitas mostrar experiencia de análisis, conviértela en casos anonimizados: dashboards, matrices, flujos de reportería, monitoreo de marca, automatización y planificación de contenido.
